Introduction to Data Science – Programa Tecnolochicas

Este repositorio reúne los proyectos y ejercicios desarrollados durante el curso Introducción a la Ciencia de Datos del Programa Tecnolochicas, en el cual participé como becaria.

Los proyectos tienen como objetivo fortalecer competencias básicas en ciencia de datos, pensamiento computacional e inteligencia artificial, a través del uso de herramientas de desarrollo visual y enfoques prácticos. El trabajo realizado prioriza el aprendizaje aplicado, la comprensión de conceptos clave y la resolución de problemas mediante el uso de datos.

Este repositorio funciona como un portafolio técnico introductorio, que documenta el proceso de aprendizaje desarrollado durante un período intensivo de formación, con énfasis en la exploración de soluciones tecnológicas con impacto educativo y social.
